Técnicas de Programação e Algoritmos

TPA, ou Técnicas de Programação, é uma disciplina onde os alunos aprendem lógica e a criação de algoritmos para resolver problemas. Por exemplo, para somar dois números, fazemos 1 + 1 = 2; porém, para colocar isso em uma linguagem de programação, utiliza-se o C++ em TPA. Eles também utilizam o Flow Algoritmo, uma ferramenta que cria fluxogramas e testa o funcionamento dos algoritmos. Com o fluxograma funcionando corretamente, o código é passado para a linguagem de programação.

Nessa disciplina, aprendem sobre algoritmos, operações lógicas e os sinais utilizados, como multiplicação (*) e divisão (/). Além disso, são introduzidos aos conceitos de verdadeiro ou falso e laços de repetição. TPA ensina a resolver problemas aplicando lógica de programação, capacitando os alunos a desenvolver soluções eficientes.
