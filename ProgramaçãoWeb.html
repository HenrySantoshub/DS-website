Programação Web

No primeiro ano, a programação web é ensinada com foco em HTML e CSS. Os alunos aprendem os fundamentos do HTML, desde a criação de títulos até a elaboração de formulários e outras funcionalidades. Complementarmente, são introduzidos aos conceitos básicos de CSS, permitindo-lhes estilizar páginas web de forma eficiente e atraente. Este conhecimento fundamental capacita os alunos a criar e personalizar websites, preparando-os para desafios mais complexos no futuro. Com uma abordagem prática e inovadora, esses estudantes estão prontos para fazer a diferença no campo da tecnologia.
